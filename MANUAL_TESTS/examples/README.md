# MANUAL_TESTS Examples

This directory contains optional example and demonstration files that showcase advanced features and development patterns.

## Contents

### Custom Phase Examples
- **`example_custom_phases.py`** - Basic examples of custom phase definitions
- **`phase_examples_demo.py`** - Interactive demonstration of phase system capabilities  
- **`test_custom_phases.py`** - Test scenarios using custom phase configurations

### Framework Development
- **`test_framework_validation.py`** - Validation tests for framework components
- **`test_improved_phases.py`** - Enhanced phase management demonstrations

## Usage

These files are optional and primarily for:
- Learning advanced configuration patterns
- Framework development and testing
- Creating custom educational scenarios
- Understanding the phase management system

Run any example directly:
```bash
cd MANUAL_TESTS/examples
python example_custom_phases.py
```

Or integrate patterns into your own custom tests in the `custom_tests/` directory.
